## 安装

您可以在[Releases选项卡](https://github.com/LNSSPsd/PhoenixBuilder/releases)中选择下载适用于您设备的预构建版本

> 如果您的运行环境基于Linux，则推荐使用较高版本的Glibc，这是因为预构建是在Ubuntu 22上完成的。
> 
> 或者，您可以选择在您的本地环境下编译本源码。
