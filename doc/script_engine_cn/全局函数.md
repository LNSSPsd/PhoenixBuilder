# 全局函数

## `printf([format][, ...args])`
* `format` `<any>`
* `...args` `<any>`

将消息进行格式化并以输出到  `stdout` **(没有换行符)**

```javascript
const count = 5;
printf("count: %d\n", count);
// Prints: count: 5, 到 stdout
```
<!-- 部分复制自 Node.JS 的文档 --> 

## `sprintf([format][, ...args])`
* `format` `<any>`
* `...args` `<any>`
* 返回值: `<string>` 格式化后的字符串
```javascript
let str=sprintf("val: %s","2");
// str = "val: 2"
```
